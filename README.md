# PhotoFinishing
该项目是基于Android平台开发的相册整理软件，根据照片拍摄的时间、地点将用户手机上面的照片整理编排展示给用户，用户可以批量整理照片，查看照片，删除照片，重命名文件夹，选择照片源、备份等操作。
